→ Open the admin folder

🔧 IF SOMETHING DOESN'T WORK:

Problem: 404 error when refreshing pages
Solution: Make sure .htaccess files are uploaded

Problem: Blank white page
Solution: Check browser console (F12), upload all files again

Problem: Admin login fails
Solution: Check browser console, verify API is running
